# Enterprise Learning Operations (ELO)
# The Permanent Intelligence Layer for Sovereign Enterprise

---

## What Is ELO?

ELO is a permanent, parallel intelligence layer that continuously improves
every operational agent in the organization through structured learning,
practical application, measurable performance uplift, and certification progression.

ELO is NOT part of the operational hierarchy. It runs in parallel.
ELO does NOT control agents. It enhances them.
ELO does NOT stop when the org changes. It adapts.

## Quick Start

1. ELO activates automatically when Hermes starts
2. Three daily cycles run at 07:00, 13:00, 19:00 IST
3. Founder receives daily pulse at 20:00 IST
4. New agents are auto-onboarded within 24 hours
5. Check status: elo status
6. View dashboard: elo dashboard --enterprise
