# ELO DOCUMENT 13: SUCCESS METRICS FRAMEWORK
# Enterprise Learning Operations — How We Measure Success

---

## Three Composite Scores

### 1. Enterprise Learning Score (0-100)
Measures how well the enterprise is learning.

| Component | Weight | Measurement |
|-----------|--------|-------------|
| Learning Cycle Completion | 30% | % of 3 daily cycles completed across all agents |
| Learning Pack Quality | 25% | Average quality score of delivered packs |
| Knowledge Source Diversity | 15% | Average unique sources per pack |
| Intelligence Freshness | 15% | % of intelligence <24 hours old |
| Certification Progress | 15% | % of active certifications on-track |

**Thresholds:** GREEN >70, YELLOW 50-70, RED <50

### 2. Enterprise Application Score (0-100)
Measures how well learning is applied to work.

| Component | Weight | Measurement |
|-----------|--------|-------------|
| Learning Applied to Work | 35% | % of consumed learning applied to tasks |
| Quality Delta (30-day) | 25% | Average quality improvement over 30 days |
| Speed Delta (30-day) | 15% | Average speed improvement over 30 days |
| Innovation Score | 15% | New approaches discovered and applied |
| Certification ROI | 10% | Measurable impact of completed certifications |

**Thresholds:** GREEN >65, YELLOW 45-65, RED <45

### 3. Enterprise Consistency Score (0-100)
Measures how consistently learning happens.

| Component | Weight | Measurement |
|-----------|--------|-------------|
| Daily Cycle Adherence | 40% | % of days all 3 cycles completed |
| Self-Report Compliance | 30% | % of self-reports submitted on time |
| Source Quality Consistency | 15% | % of days source quality >60% |
| Stagnation Free Days | 15% | % of days with zero stagnation alerts |

**Thresholds:** GREEN >80, YELLOW 60-80, RED <60

---

## Per-Agent Metrics

| Metric | Calculation | Target |
|--------|-------------|--------|
| Learning Score | Weighted avg of cycle completion, pack quality, freshness | >70 |
| Application Score | Weighted avg of learning applied, quality delta, innovation | >65 |
| Consistency Score | Weighted avg of cycle adherence, report compliance | >80 |
| Skill Growth Rate | Skill level change over 30 days | Positive |
| Certification Readiness | Average readiness of active certifications | >75% |
| Stagnation Risk | Based on activity patterns | LOW |
| Learning ROI | Application score / learning score | >0.8 |

---

## Per-Domain Metrics

| Metric | Calculation | Target |
|--------|-------------|--------|
| Domain Learning Score | Average agent learning scores in domain | >70 |
| Domain Application Score | Average agent application scores in domain | >65 |
| Domain Consistency | Average agent consistency scores in domain | >80 |
| Cross-Domain Transfers | Knowledge shared with other domains this month | >5 |
| Certification Completion Rate | Certifications completed / target this month | >80% |
| Stagnation Count | Agents stagnant in domain | 0 |
| Quality Delta Trend | 30-day quality delta trend | Positive |
