## Automation Scope

**Before Each Session (30min pre-work)**
1. Collect recent ADRs from domain (past 2 weeks)
2. Summarize quality delta per domain member
3. Identify common failure patterns (from gaming detection + post-mortems)
4. Generate draft agenda with suggested discussion topics
5. Distribute pre-read to participants

**During Session (automated facilitation)**
1. Track discussion topics against agenda
2. Capture decisions to decision log
3. Flag action items with owners
4. Generate real-time minutes
5. Ensure all registered participants contribute

**After Session (automated follow-up)**
1. Publish minutes to knowledge store
2. Create action items in tracking system
3. Update any referenced ADRs
4. Schedule follow-up for incomplete actions
5. Track action closure rate per CoP

## Effectiveness Metrics

| Metric | Target | Measurement |
|--------|--------|-------------|
| Decision implementation rate | >80% within 30 days | Decision log to action completion |
| Cross-domain problem reduction | >20% fewer recurring issues | Incident classification trends |
| Knowledge distribution breadth | >60% of domains represented | Attendance records |
| Action completion on time | >85% | Action item tracking |
