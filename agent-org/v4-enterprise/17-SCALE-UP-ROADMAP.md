## Scaling Principles

1. **Hire agents, not roles** — Each agent fills a role; roles don't create agents
2. **Quality before quantity** — Never sacrifice quality for speed
3. **Automate before hiring** — Automate repetitive tasks before adding agents
4. **Shared platform** — Build shared infrastructure early, leverage later
5. **Standards first** — Establish standards before scaling
6. **Measure everything** — You can't improve what you don't measure
7. **Iterate on process** — Process evolves with the organization
8. **Keep it simple** — Complexity is the enemy of scaling
